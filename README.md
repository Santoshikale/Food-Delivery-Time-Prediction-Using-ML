## 🍔 Food Delivery Time Prediction Using Machine Learning

## 📌 Project Overview
This project predicts the estimated food delivery time using Machine Learning based on factors such as:
- Distance
- Weather
- Traffic Level
- Time of Day
- Vehicle Type
- Preparation Time
- Courier Experience

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## 🚀 Features
- Predict delivery time
- User-friendly Streamlit interface
- Machine Learning model
- Responsive input form
